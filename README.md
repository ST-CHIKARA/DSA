# DSA

This repository documents my journey learning Data Structures and Algorithms.

I am following [freecodecamp DSA with python](https://www.youtube.com/watch?v=pkYVOmU3MgA) & [Algorithms and Data Structures tutorial](https://youtu.be/8hly31xKli0?si=W2lij_ccIcp08QCV) youtube videos for this. 

However, I realized that simply watching the videos wasn’t enough for me. I prefer understanding how data flows, how things start, transform, and reach an outcome in my own understanding.

Because of that, I created this repository to break down concepts in my own way and build a deeper understanding. If you also like learning by truly understanding, you might find this helpful so feel free to follow along
